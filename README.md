# mom-is-coming-home
A chaotic 1-hour hackathon project that won 3rd place! Mom is 5 mins away and the house is a wreck. Race to clean spilled Milo, hide failed exams, and survive a mocking AI assistant before the final inspection. Will you get S-Rank or face the slipper? Single-file HTML/JS with emoji-based graphics. 🎮✨
